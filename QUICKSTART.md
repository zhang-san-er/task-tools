# 快速开始指南

## 安装和运行

1. **安装依赖**
   ```bash
   npm install
   ```

2. **启动开发服务器**
   ```bash
   npm run dev
   ```

3. **在浏览器中打开**
   访问 `http://localhost:5173`（或终端显示的地址）

## 生成图标（可选）

1. 在浏览器中打开 `public/icons/generate-icons.html`
2. 点击下载按钮保存图标文件
3. 将下载的文件重命名并放置到 `public/icons/` 目录：
   - `icon-192x192.png`
   - `icon-512x512.png`

## 构建生产版本

```bash
npm run build
```

构建完成后，`dist` 目录包含所有静态文件，可以：
- 部署到静态文件服务器
- 打包为 ZIP 文件供用户下载

## 功能说明

### 主线任务
- 完成任务获得积分
- 积分累积可升级
- 未完成任务会扣除少量生命值

### 心魔挑战
- 完成任务获得高额积分（额外50%奖励）
- 成功抵御心魔可恢复生命值
- 失败会扣除较多生命值

### 用户系统
- **等级**：每100积分升一级
- **生命值**：0-100，归零会有警告
- **连续天数**：连续完成任务的天数

### 数据管理
- 点击右下角设置按钮
- 可以导出/导入数据备份
- 可以清除所有数据

## 注意事项

- 所有数据存储在浏览器 localStorage
- Service Worker 需要 HTTPS 或 localhost 环境
- 清除浏览器数据会丢失所有记录
- 建议定期导出数据备份
